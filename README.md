# 📝 Java Text Editor  

A simple **text editor** built using Java Swing. Supports basic file operations, editing, and printing.  

## ✨ Features  
✅ Open & Save text files  
✅ Cut, Copy, Paste functionality  
✅ Print documents  
✅ Simple GUI using Swing
